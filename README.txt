<h2>BAS - BRAMS Automation Suite</h2>

Os modelos numéricos regionais são usados como ferramentas na geração de previsões de tempo e que trazem uma maior precisão se comparado aos modelos globais, já que são utilizados em regiões limitadas. Atualmente o CPTEC/INPE (Centro de Previsão de Tempo e Estudos Climáticos do Instituto Nacional de Pesquisas Espaciais) é o órgão responsável pelo desenvolvimento contínuo do BRAMS (Brazilian Regional Atmospheric Modelling System), um modelo regional preparado para ser aplicado às condições tropicais. Mesmo sendo um modelo continuamente em desenvolvimento, muitas tarefas ainda são necessárias para difundir e facilitar sua utilização em um número maior de centros de pesquisa, universidades e na iniciativa privada.

Uma tarefa relevante para o CPTEC/INPE é desenvolver uma suíte operacional responsável pela execução do modelo desde a fase de pré-processamento até a geração dos resultados finais em uma forma que possa ser adaptável para regiões específicas, utilizando uma linguagem de programação robusta e bastante difundida internacionalmente.

A proposta do presente trabalho é utilizar a linguagem python para o desenvolvimento de uma suíte operacional para servir a Região do Vale do Paraíba com previsões de altíssima resolução espacial. O phyton tem sido amplamente utilizado na área de meteorologia nos últimos anos e aplicá-lo na construção do ambiente é altamente desejável. As etapas de desenvolvimento consistem na aquisição de dados (arquivos de entrada) para o pré-processamento, rodar o modelo e pós-processar os resultados para gerar imagens e/ou informações referentes às variáveis de saída do modelo.
